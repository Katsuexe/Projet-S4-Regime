# Schema de Base de Donnees

Le schema comprend 7 tables principales :

- `users` : Utilisateurs avec statut Gold
- `user_sante` : Donnees IMC et objectifs
- `regimes` : Catalogue des regimes alimentaires
- `regime_durees` : Prix selon duree d'abonnement
- `activites` : Activites sportives complementaires
- `codes` : Codes promo pour portefeuille virtuel
- `user_regimes` : Historique des abonnements

Voir le fichier script.sql pour le schema complet et les donnees d'exemple.
